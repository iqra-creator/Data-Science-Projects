# Data Science Projects by Iqra Masood

Welcome to my **Data Science Projects repository**! This collection showcases real-world analytics, predictive modeling, machine learning, and data visualization projects completed using Python, SQL, and various data analysis tools. Each project is structured to demonstrate **data preprocessing, feature engineering, modeling, evaluation, and insights generation**.

## Projects Overview

### Task:1 Mall Customers Segmentation
- **Objective:** Segment mall customers based on purchasing behavior and demographics.
- **Tech Stack:** Python, Pandas, Scikit-learn, Matplotlib, Seaborn
- **Key Features:**
  - Data preprocessing and cleaning
  - Exploratory Data Analysis (EDA)
  - Customer segmentation using K-Means clustering
  - Visualizations of clusters and customer profiles
- **Impact:** Helps businesses target marketing campaigns effectively based on customer segments.

### Task:2 Predicting House Prices
- **Objective:** Predict housing prices based on property features.
- **Tech Stack:** Python, Pandas, Scikit-learn, Matplotlib
- **Key Features:**
  - Feature engineering (location, size, amenities)
  - Handling missing values and outliers
  - Linear regression and cross-validation
  - Performance evaluation using RMSE and MAE
- **Impact:** Provides actionable insights for real estate investment and pricing strategies.

### Task:3 Sentiment Analysis of Social Media Posts
- **Objective:** Analyze sentiment of social media posts to understand public perception.
- **Tech Stack:** Python, NLP, VADER, BERT, Pandas, Matplotlib
- **Key Features:**
  - Data collection and cleaning
  - Text preprocessing (tokenization, stop words removal, lemmatization)
  - Sentiment classification (positive, negative, neutral)
  - Trend visualizations
- **Impact:** Supports businesses in monitoring brand reputation and customer feedback.

### Task:4 Anomaly Detection in Network Traffic
- **Objective:** Detect anomalies in network traffic for cybersecurity purposes.
- **Tech Stack:** Python, Pandas, Scikit-learn, Isolation Forest, Autoencoders
- **Key Features:**
  - Data preprocessing and feature engineering
  - Implementation of Isolation Forest and Autoencoders
  - Evaluation using ROC-AUC
  - Alerts and visualizations for anomaly patterns
- **Impact:** Helps IT teams proactively identify suspicious activity and security threats.

### Task:5 Heart Disease Prediction
- **Objective:** Predict likelihood of heart disease from patient health data.
- **Tech Stack:** Python, Pandas, Scikit-learn, Logistic Regression
- **Key Features:**
  - Feature selection based on domain knowledge
  - Model training and evaluation (accuracy, precision, recall)
  - Hyperparameter tuning
  - Model interpretability using SHAP values
- **Impact:** Assists healthcare professionals in early detection and clinical decision-making.

### Task:6 Coffee Shop Sales Analysis
- **Objective:** Analyze coffee shop sales and forecast future revenue.
- **Tech Stack:** Python, Pandas, Matplotlib, Seaborn, ARIMA
- **Key Features:**
  - Data cleaning and preparation
  - Exploratory Data Analysis (EDA)
  - Sales forecasting using ARIMA
  - Insights on product performance and profitability
- **Impact:** Guides business decisions for inventory, promotions, and revenue optimization.

## Tools & Technologies Used
- **Programming & Analysis:** Python, Pandas, NumPy, Scikit-learn
- **Visualization:** Matplotlib, Seaborn
- **Data Sources:** CSV, SQL databases
- **Machine Learning:** Clustering, Regression, Classification, Anomaly Detection
